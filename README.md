# Black Soldier Fly (Hermetia illucens) Image Collection
A curated collection of high-quality Black Soldier Fly (*Hermetia illucens*) animated thumbnails and lifecycle figure covering all lifestages.

## Purpose
This repository has been created to support scientific research, education, industry outreach, and public communication relating to the Black Soldier Fly.
Users are encouraged to use these images in publications, presentations, reports, websites, teaching materials, and other non-commercial or commercial applications, provided appropriate attribution is given.

## Image Contents
The repository may include:
* Eggs
* Larvae
* Prepupae
* Pupae
* Adults
* Lifecycle figure
